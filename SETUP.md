# Directory Structure Setup

After downloading these files, create the following folders:

## Required Folders

1. **images/** - Store all your images here
   - profile.jpg (your profile photo)
   - project1.jpg, project2.jpg, etc. (portfolio images)
   - Any blog post images

2. **blog-posts/** - Already created with post1.html template
   - Add more blog post HTML files here

3. **documents/** - Store downloadable files
   - cv.pdf (your CV as a PDF)
   - Any other documents you want to share

## Folder Structure

```
your-repository/
├── index.html
├── blog.html
├── portfolio.html
├── cv.html
├── styles.css
├── script.js
├── README.md
├── .gitignore
├── images/
│   ├── profile.jpg
│   ├── project1.jpg
│   ├── project2.jpg
│   └── ...
├── blog-posts/
│   ├── post1.html
│   ├── post2.html
│   └── ...
└── documents/
    └── cv.pdf
```

## Image Guidelines

### Profile Photo
- Recommended size: 400-800px wide
- Format: JPG or PNG
- Keep file size under 500KB for faster loading

### Portfolio Images
- Recommended size: 1000-1500px wide
- Format: JPG, PNG, or WebP
- Compress images for web use
- Keep each file under 1MB

## Next Steps

1. Create the `images/` folder
2. Add your profile photo as `images/profile.jpg`
3. Add your portfolio images
4. Create the `documents/` folder
5. Add your CV as `documents/cv.pdf`
6. Update all HTML files with your information
7. Test locally by opening index.html in a browser
8. Push to GitHub when ready
